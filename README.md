# Unité 4 — La protection animale

Web para estudiar la Unité 4 de francés: vocabulario, subjonctif présent, *ne… que*, obligación e interdicción.

## Modos

- **Vocabulario** filtrable con audio (TTS del navegador).
- **Gramática** explicada.
- **Subjuntivo ★** — disparadores, 4 ejercicios distintos.
- **Flashcards** con sistema Leitner (guardado local).
- **Quiz** de opción múltiple.
- **Escribir** (traducción libre tolerante).
- **Conjugar** subjonctif présent.
- **Hacer examen ★** — examen digital autocorregido con base de respuestas aceptables.
- **Examen PDF** — examen imprimible con hoja de respuestas.

## Desarrollo

Es estático: HTML + CSS + JS, sin build. Abre `index.html` en el navegador o sirve con cualquier servidor estático.

```bash
python3 -m http.server 8000
# http://localhost:8000
```

## Despliegue en GitHub Pages

El workflow `.github/workflows/pages.yml` despliega automáticamente al rama `main`. En Settings → Pages, elige **Source: GitHub Actions**.
